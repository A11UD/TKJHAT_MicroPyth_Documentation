# VEML6030

## Detailed Description 
This module provides API for driving Vishay VEML6030 ambient light sensor in the TKJHAT extencion board. VEML6030 is connected to the I2C bus with address 0x10.
 
### It’s configured with:
- gain: 1/8
- Integration time: 100ms
- Persisentec protect number setting: 1
- interrupts: disable
- Power: on


**Note:** The VEML6030 supports alert thresholds and an interrupt pin, but interrupt handling is not implemented in this MicroPython driver.

### Default connections (TKJHAT)

| Signal              | MicroPython / Board              | GPIO | Description                               |
|---------------------|----------------------------------|------|-------------------------------------------|
| SDA                 | I2C(..., sda=Pin(12))             | 12   | I2C data                                  |
| SCL                 | I2C(..., scl=Pin(13))             | 13   | I2C clock                                 |
| Address             | 0x10                             | –    | VEML6030 I2C address                       |
| Interrupt (optional)| not used                         | 9   | VEML6030 alert pin (not implemented)       |




### Usage example:
**Precondition:** The I2C interface must be initialized in your main.py before using the driver.

```python
from machine import I2C, Pin
from hdc2021 import VEML6030
import time
i2c = I2C(0, sda=Pin(12), scl=Pin(13), freq=400_000)

sensor = VEML6030(i2c)
sensor.init()

while True:
    try:
        sleep_ms(500)
        ambient_light = sensor.read()
        print(ambient_light)

    except KeyboardInterrupt:
        break
```


## VEML6030 code documentation
class VEML6030(i2c) creates ambient light sensor instance using existing i2c bus.


## Function Documentation


# `init()`

### What it does:
Sets the configuration of the VEML6030 ambient light sensor and waits 10 ms.


### Configures:
- gain: 1/8
- Integration time: 100ms
- Persisentec protect number setting: 1
- interrupts: disable
- Power: on

### Precondition
- I2C bus must be initialized (SDA = 12, SCL = 13, freq = 400 kHz)

### Postcondition
- The sensor is configured and ready to measure ambient light.  
  

# `read() -> float`
Read ambient light in lux.    

### What it does: 
Fetches the raw ALS (ambient light sensing) output and applies the recommended conversion factor. For high-light conditions, a polynomial correction is applied (more info about the correction at: https://www.vishay.com/docs/84367/designingveml6030.pdf).

### Returns
- Ambient light in lux.

### Notes
- Sampling time should be > Integration time (100ms). Reading values faster than that may return same value multiple times.  
  
# `stop()`
Power off the veml6030 ambient light sensor.     

### What it does 
- Set the power bit in the configuration to 1.  
- Wait 10 ms

### Notes  
- Reading values after powering off the sensor returns the last recorded value.
