# HDC2021

## Detailed Description 
This module provides a MicroPython API for Texas Instruments HDC2021 Digital temperature and humidity sensor that is used on the TKJHAT extension.
The HDC2021 is connected with the I2C bus at address 0x40 and it provides temperature and humidity measurements. This drivers configures the sensor to work with the
MicroPython on RP2040.
 
**It’s configured with:**

- Continuous measurement mode (temperature and humidity).
- Sampling rate 1 Hz.
- Resolution: 14-bit for both measurements.
- Optional threshold configuration

**Note:** The HDC2021 supports alert thresholds and an interrupt pin, but interrupt handling is not implemented in this MicroPython driver. Threshold functions are included for compatibility and future use.

### Default connections (TKJHAT)

| Signal              | MicroPython / Board              | GPIO | Description                               |
|---------------------|----------------------------------|------|-------------------------------------------|
| SDA                 | I2C(..., sda=Pin(12))             | 12   | I2C data                                  |
| SCL                 | I2C(..., scl=Pin(13))             | 13   | I2C clock                                 |
| Address             | 0x40                             | –    | HDC2021 I2C address                       |
| Interrupt (optional)| not used                         | 21   | HDC2021 alert pin (not implemented)       |



### Default thresholds are configured to:

- Temperature low:				 - 30°C
- Temperature high:				 + 50°C
- Humidity low: 					 0 %RH
- Humidity high:					100 %RH


### Usage example:
**Precondition:** The I2C interface must be initialized in your main.py before using the driver.

Example 1: Basic usage
Example 2: Multiple devices with BusManager

```python
#Example 1
from machine import I2C, Pin
from hdc2021 import HDC2021
import time
i2c = I2C(0, sda=Pin(12), scl=Pin(13), freq=400_000)
sensor = HDC2021(i2c)
sensor.init()
while True:
    temp = sensor.read_temperature()
    humid = sensor.read_humidity()
    print(temp, humid)
    time.sleep(1)   # sensor is configured for 1 Hz

#Example 2: Multiple devices usage
from bus_manager import I2CBusManager
from machine import I2C, Pin
from hdc2021 import HDC2021
import time

i2c = I2C(0, sda=Pin(12), scl=Pin(13), freq=400_000)
bus = I2CBusManager(i2c)

sensor = HDC2021(bus)
sensor.init()
```
### Notes: 
- The sensor is configured to 1 Hz sampling, meaning the value updates once per second. 
- Reading more frequently may return repeated values.
- Temperature and humidity are read from 16-bit registers:
 - Temperature: TEMP_LOW (0x00) and TEMP_HIGH (0x01)
 - Humidity: HUMID_LOW (0x02) and HUMID_HIGH (0x03)
- Conversion formulas used(from datasheet):
```python
**Humidity (%RH):** = (HUMIDITY[15:0] / 2^16) × 100

**Temperature (°C):**  = (TEMPERATURE[15:0] / 2^16) × 165 − 40
```

## HDC2021 code documentation
**class HDC2021(i2c)** 
Create an HDC2021 sensor instance using:

- machine.I2C
- or I2CBusManager (recommended for multi-device setups)
- The driver is compatible with both standard I2C and the centralized I2CBusManager.


## Function Documentation


## `init()`

**What it does:**

- Performs a soft reset (CONFIG |= 0x80) and waits 50 ms

**Configures:**

  - Measurement mode: temperature + humidity
  - Sampling rate: 1 Hz
  - Resolution: 14-bit temperature, 14-bit humidity
- Sets default thresholds:
  - Temperature: -30 °C / +50 °C
  - Humidity: 0 % / 100 %
- Triggers continuous measurement

**Precondition**
- I2C bus must be initialized (SDA = 12, SCL = 13, freq = 400 kHz)

**Postcondition**
- Sensor is measuring continuously and values can be read


## `read_temperature() -> float`
Read temperature in Celsius.

**Returns**
- Temperature as a floating-point value in °C

**Notes**
- The sensor is configured for 1 Hz sampling.
- Reading faster than 1 second may return the same value multiple times.



## `read_humidity() -> float`

Read relative humidity in %RH.

**Returns**
- Relative humidity as a floating-point value in percent (0–100)

**Notes**
- The sensor is configured for 1 Hz sampling.

## HDC2021 API documentation

## `set_low_temp_threshold(temp_c: float) -> None`

Set the low temperature threshold for the alert system.

**Parameters**
- `temp_c`: Temperature in °C (−40…125)

**Notes**
- Interrupt output is not implemented, but the threshold register is configured.



## `set_high_temp_threshold(temp_c: float) -> None`

Set the high temperature threshold for the alert system.

**Parameters**
- `temp_c`: Temperature in °C (−40…125)


## `set_low_humidity_threshold(humidity: float) -> None`

Set the low humidity threshold for the alert system.

**Parameters**
- `humidity`: Relative humidity in %RH (clamped to 0–100)



## `set_high_humidity_threshold(humidity: float) -> None`

Set the high humidity threshold for the alert system.

**Parameters**
- `humidity`: Relative humidity in %RH (clamped to 0–100)


## `stop() -> None`

Stop continuous measurements and reduce power usage.

**What it does**
- Disables automatic measurement mode bits (AMM)
- Disables heater and interrupt output
- Clears measurement trigger bit

**Notes**
- Calling `init()` again will reconfigure the sensor and restart measurements.
